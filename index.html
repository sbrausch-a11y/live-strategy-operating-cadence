Rules of engagement

Use these rules to resolve timing conflicts, protect pre-reads, and keep follow-up ownership clear.

Live-only cadences

Use the Live-only prep rhythm. No Core Submission Deadline or enterprise handoff is required.

Core cadences

Core Submission is the deadline for Live input into the Core process. Live Refresh is the latest Live read before the forum.

MBR replacement rule

On Jamie MBR meeting weeks, there is no Jamie WBR. Starting in July, Live MBR replaces Live Global WBR during the MBR materials-due week so the Live team can do month-close readout ahead of Core lock.

Jamie WBR

Live VP Review happens Monday night. Core Submission is due Tuesday by noon; Meeting is Wednesday.

Jamie MBR

Core Submission and Core Pre-Read Lock are firm. Live Refresh happens 48 hours before the meeting unless otherwise aligned.

MOSH

Live Refresh target is 48 hours before the meeting. If the refresh misses the pre-read path, Live shares after the call.
